# AFD Reading Guide and Assignment Schedule

This guide provides a day-by-day reading schedule across the first three weeks of the program. Each day is labeled by the day you should have the material read by. If it says **Tuesday**, have it read by Tuesday.

## File Types

- **LEC** -- Module lecture document. Comprehensive reference for the module.
- **CT** -- Concept thread. Focused reading on a single topic.
- **RV** -- Review material for exam prep.
- **Notebook** (.ipynb) -- Hands-on workshop. Work through alongside the lecture and CTs.

## How Links Work

Links to existing content will open the file directly. Links to planned content (modules 005-010 under SageMaker, AWS Bedrock) will resolve once those files are added to the AFD directory.
